To write a Java program using control statements (if, loops) to check whether a number is prime or not. 
 
ALGORITHM: 
Step 1: Start the program  
Step 2: Input a number 
Step 3: If number ≤ 1, it is not prime 
Step 4: Use loop from 2 to n/2 to check divisibility  
Step 5: If divisible, it is not prime 
Step 6: Else, it is prime  
Step 7: Display result  
Step 8: Stop the program 
 
PROGRAM: 
import java.util.Scanner; class Main { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int n, count = 0; 
System.out.print("Enter a number: "); n = sc.nextInt(); for (int i = 1; i <= n; i++) { if (n % i == 0) { count++; 
} 
} 
if (count == 2) 
System.out.println("Prime Number"); else 
System.out.println("Not a Prime Number"); 
} 
}

AIM: 
To read data from a text file and write it to another file using FileReader and FileWriter. 
 
ALGORITHM: 
           Step 1: Start the program 
           Step 2: Create FileReader to read input file 
           Step 3: Create FileWriter to write into output file 
           Step 4: Read data character by character 
           Step 5: Write data into new file 
           Step 6: Close both files 
           Step 7: Stop the program 
 
PROGRAM: 
import java.io.*; class Main {     public static void main(String[] args) {         try { 
            FileReader fr = new FileReader("input.txt");             FileWriter fw = new FileWriter("output.txt");             int ch; 
            while ((ch = fr.read()) != -1) {                 fw.write(ch); 
            } 
 
            fr.close();             fw.close(); 
            System.out.println("Data copied successfully"); 
        } catch (Exception e) { 
            System.out.println("Error"); 
        } 
    } 
}
